## 一、计算机网络概述

1. **定义**：地理位置不同的独立主机，通过通信线路连接实现资源共享
2. **组成**：资源子网 + 通信子网
3. **拓扑分类**：星型、总线型、网状型、环型、树型
4. **范围**：LAN（局域网）、MAN（城域网）、WAN（广域网）
5. **传输方式**：有线、无线
6. **方向分类**：单工、半双工、全双工
7. **传输类型**：单播、组播、广播
8. **数据交换方式**：电路交换、报文交换、分组交换
9. **协议三要素**：语法、语义、时序（同步）
10. **OSI 七层模型**：应用层 → 表示层 → 会话层 → 传输层 → 网络层 → 数据链路层 → 物理层
11. **TCP/IP 四层模型**：网络接口层 → 网络层 → 传输层 → 应用层

## 二、物理层基础概念

1. **四大特性**：机械特性、电气特性、功能特性、过程特性
2. **两种信号**：数字信号（离散）、模拟信号（连续）
3. **调制与编码**
   - 调制：数字信号 → 模拟信号
   - 编码：模拟信号 → 数字信号（采样 → 量化 → 编码）
4. **传输介质**
   - 双绞线：约 100m
   - 光纤：单模（几十千米）、多模（几千米）
5. **通信系统模型**：信源 → 发送设备 → 信道 → 接收设备 → 信宿
6. **信道复用技术**
   - 频分复用（FDM）、时分复用（TDM）、波分复用（WDM）、码分复用（CDM）
   - 复用目的：同一信道传输多路信号，互不干扰
7. **数据传输方式**
   - 按同时传输数量：串行传输、并行传输
   - 按收发双方行为：同步传输、异步传输
   - 按信号类型：基带传输（数字信号）、频带传输（模拟信号）
8. **物理层设备**：中继器（信号放大）、集线器（多端口中继器）
9. **两大定理**
   - 奈奎斯特定理：理想低通信道，最大传输速率 = \(2W \log_2 V\)
   - 香农定理：有噪声信道，最大传输速率 = \(W \log_2(1 + S/N)\)
10. **传输方向**：单工、半双工、全双工
11. **传播对象**：单播、组播、广播

## 三、数据链路层

1. **数据链路层概念**：为相邻节点提供可靠数据传输
2. **帧概念**：帧是数据链路层的单位
   - 组成：帧头 | 数据 | 帧尾
   - 帧头：源MAC地址、目的MAC地址、控制、类型
3. **以太网数据帧**：将数据链路层分为 MAC 和 LLC 两个子层
   - **MAC**：对接物理层，介质访问控制，屏蔽不同物理链路的差异
   - **LLC**：对接网络层，逻辑链路控制，实现数据包的分段和重组
4. **链路层传输方式**：单播、广播

### 5. 数据链路层的过程

1. 封装成帧 —— PPP 协议
2. 透明传输
3. 差错检测 —— CRC 协议

### 6. 以太网中的设备

- 集线器
- 交换机
- 网桥

### 7. 冲突域和广播域

- 交换机的每一个接口，都是一个冲突域
- 所有接口组成的一个域，叫广播域

**区别：**

1. 广播域可以跨网段
2. 冲突域基于物理层，广播域基于数据链路层
3. HUB 所有端口都在一个广播域，也是一个冲突域
4. 同一冲突域共享带宽

### 8. VLAN 虚拟局域网

将一个物理的局域网划分成为多个虚拟的局域网。

**优点：**

1. 划分广播域
2. 增强局域网的安全性
3. 提高健壮性
4. 灵活构建工作组

### 9. CSMA/CD

广播型信道中的防冲突技术，适用于总线型网络。

**四大特点：**

1. 先听再发
2. 边听边发
3. 冲突停止
4. 延迟后发

### 10. PPP 协议

用于点对点链路，支持单播，负责封装数据。

### 11. CRC 循环冗余校验

用于差错检测。

## 四、网络层

1. **网络层的作用**：提供点到点的服务，单位：分组
2. **网络层协议**
   - **ARP**：地址解析协议 —— 根据IP地址获取物理地址
   - **ICMP**：网际控制报文协议
   - **IGMP**：网际组管理协议
3. **IP 地址**
4. **子网划分**
5. **路由**：指分组从源到目的地时，决定端到端路径的网络范围进程。路由是指导报文转发的路径信息，通过路由可以确认转发IP报文的路径。
6. **路由器**：网络层的基本设备，负责数据转发。一个端口代表一个网段，路由器中存放着通往各个网段的表格，叫路由表。
7. **路由表**：一个存储在路由器或联网计算机中的电子表格（文件或类数据库），路由表存储着指向特定网络地址的路径。
8. **网关**：又称网间连接器、协议转换器，用于两个高层协议不同的网络连接。网关既可以用于广域网互联，也可以用于局域网互联。
9. **路由获取方式**：直连路由、静态路由、动态路由

### 10. 路由的配置方式

#### 静态路由

由管理员手工配置，配置方便。

- **静态路由**：手动指定的固定路由
- **缺省路由**：当报文没有在路由表中找到匹配的具体表项时使用的路由

#### 动态路由

由协议自身的路由算法自动计算，能够自动适应网络拓扑的变化，适用于具有一定数量三层设备的网络。

常见协议：**RIP** 和 **OSPF**

| 协议 | RIP | OSPF |
|------|-----|------|
| 类型 | 基于距离矢量的动态路由协议 | 基于链路状态的协议 |
| 适用规模 | 中小型网络拓扑 | 较大规模的网络 |
| 子网掩码 | 不支持可变长度子网掩码 | 支持可变长度子网掩码 |
| 收敛速度 | 较慢 | 更加迅速 |
| 防环 | 不防环 | 防环 |
